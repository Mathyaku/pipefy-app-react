# pipefy-react

Pipefy - Application to control your tasks using Drag and Drop functionality.

Made using React framework and NodeJS

**Back-End**

***install dependencies**: yarn install <br />
**run cmd**: yarn dev <br />
**port**: 3333*

**Front-End Web**

***install dependencies**: yarn install <br />
**run cmd**: yarn start <br />
**port**: 3000*
